# dotfiles

Setup steps to follow:
- Follow [this guide](https://docs.github.com/en/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent) to setup ssh key
- Download Jetbrains IDE, copy over `jetbrains/.ideavimrc` to home directory, and import settings from `jetbrains` directory
- Run `init` bash script
